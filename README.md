# Task Manager App (React Native)

This is a simple **Task Manager mobile application** built with **React Native**.

The app allows users to:
- Add tasks
- Set a start date and end date
- View all tasks
- Delete tasks when finished

This project is made for **beginners** who are learning React Native basics.

---

## Features

- Add a task with:
  - Title
  - Description
  - Start date
  - End date
- Display tasks in a list
- Delete tasks
- Select dates using a date picker
- Simple and clean user interface

---

## Technologies Used

- React Native
- React Hooks (`useState`)
- DateTime Picker (`@react-native-community/datetimepicker`)
- JavaScript (ES6)
